# Treasure Labyrinth

Text adventure game "Treasure Labyrinth" - a pet project.

## Installation

```bash
make install
```

## Running

```bash
make project
```

or

```bash
poetry run project
```

## Development

### Code linting

```bash
make lint
```

### Building package

```bash
make build
```

### Test publication (dry-run)

```bash
make publish
```

### Installing package to system

```bash
make package-install
```
